Customizing Clone

Not a fan of Brave, Clone's default theme? No problem!

Clone has a set of built-in variables that you can customize through Sass that will allow you to control a variety of fonts, colours, whitespace, etc. Simply pick and choose the variables you want to customize from the list below and add them to your Sass file, directly above Clone's import function (don't forget to remove the !default):

// My Custom Clone Theme
// Please note that Clone must be included last, after all of your custom variables.

// Clone Theme Variables
// All available theme variables can be found at https://github.com/joshdrink/clone-framework/blob/master/app/scss/core/_defaults.scss
$clone-grid-columns: 8;
$clone-c1: #5B1647;

// Include Clone
@import "../node_modules/clone-framework/app/scss/clone";

[data-clone] {
    // Your custom CSS goes here.
}
// Clone =======================================================================

// Layout Variables
$clone-large-layout: 80rem;
$clone-medium-layout: 60rem;
$clone-small-layout: 39rem;

// Whitespace Variables
// On Portrait Phone Devices, Media Queries exist at Tablet Landscape, and Desktop Large and are generated using the multiplier below.
$clone-margin: 1rem;
$clone-padding: 1rem;
$clone-whitespace-multiplier: 1.4;

// Grid Variables
$clone-grid-columns: 12;
$clone-grid-gutter: $clone-padding;

// Main Colour Variables

    // Black & White
    $clone-white: #FFFFFF;
    $clone-black: rgba(10, 10, 20, 1);

    // gray/Gray
    $clone-gray: rgba(150, 150, 150, 1);

    // Theme Colours
    $clone-c1: #F14948;
    $clone-c2: #F1604D;
    $clone-c3: #F47953;
    $clone-c4: #F79559;
    $clone-c5: #F7AF5B;

    // Go, Slow, Stop
    $clone-go: green;
    $clone-slow: orange;
    $clone-stop: red;
    $clone-error: $clone-stop;

// Sitewide Colour Variables

    // Text Selection
    $clone-selection-background-colour: $clone-c5;

    // Hover
    $clone-hover-colour: $clone-c4;

    // Focus
    $clone-focus-colour: $clone-hover-colour;

    // Visited Links
    $clone-visited-colour: purple;

// Component Options

    // Cards
    $clone-card-shadow: 0 3px 5px rgba(0, 0, 0, .5);

// Component Colour Variables

    // Accordions
    $clone-accordion-alternating-background-colour: lighten($clone-gray, 35%);

    // Alerts
    $clone-alert-information: $clone-go;
    $clone-alert-warning: $clone-slow;
    $clone-alert-error: $clone-stop;

    // Blockquotes
    $clone-blockquote-colour: $clone-c1;

    // Carousels
    $clone-carousel-dot-colour: $clone-gray-light;
    $clone-carousel-dot-active-colour: $clone-hover-colour;

    // Code Blocks
    $clone-code-background: $clone-black;

    // Menus
    $clone-menu-background-colour: $clone-black;
    $clone-submenu-background-colour: $clone-c2;

    $clone-menu-font-colour: $clone-white;
    $clone-menu-divider-colour: rgba($clone-c4, .5);
    $clone-menu-icon-colour: $clone-white;

    // Pullquotes
    $clone-pullquote-colour: $clone-c2;

    // Tables
    $clone-table-heading-colour: $clone-c1;
    $clone-table-alternating-colour: rgba(0, 0, 0, .1); // Note that this transparency allows for a checker pattern on the grid when there is a vertical heading column,.

// Font Variables

    // Fonts
    $clone-font-sans: "Open Sans", sans-serif;
    $clone-font-serif: serif;
    $clone-font-script: "Euphoria Script", cursive;
    $clone-font-monospace: "Consolas", monospace;

    $clone-font-default: $clone-font-sans;

    // Default Line Height
    $clone-line-height: 1.5;

    // The Typography Scale
    $clone-default-font-size: 1rem;
    $clone-font-scale: 1.333;

Pre-built Themes

Brave Theme
Preview Brave

Brave is the default theme installed on Clone. It uses a progression of a soft orange-red to orange-yellow to give the theme a warm, upbeat vibe.

Canada.ca Theme
Preview Canada.ca

The Canada.ca theme is built to emulate the basic style (colour, font, whitespace, etc.) of the Canada.ca government theme.

// Clone: Canada.ca Edition ====================================================

// Layout Variables
$clone-large-layout: 100rem;
$clone-medium-layout: 60rem;
$clone-small-layout: 39rem;

$clone-padding: 1rem;

$clone-grid-columns: 12;
$clone-grid-gutter: $clone-padding;

    // Focus

    // Visited Links

// Component Options

    // Cards

// Component Colour Variables

    // Accordions

    $clone-accordion-alternating-background-colour: lighten($clone-gray, 35%);

    // Alerts

    // Blockquotes

    // Carousels

    $clone-carousel-dot-active-colour: $clone-c3;

    // Code Blocks
    $clone-code-background: $clone-black;

    // Menus
    $clone-menu-background-colour: $clone-white;

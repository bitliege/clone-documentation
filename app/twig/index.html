Welcome to Clone (beta)

Clone is a simple design framework for building websites.

It leverages the power of HTML5 data attributes to provide easy-to-use and customizable properties. Clone can be used as is or it can be imported into your project to allow for deep customization through the power of Sass.

Browser Support: IE11, Edge, Chrome, Firefox, Safari

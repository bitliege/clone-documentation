Font Family

<div data-c-font="STYLE"></div>

This attribute allows you to assign a specific font-family to an element. Font family arguments can be customized using the variables found in the themeing section of the documentation.

Available Arguments 
# STYLE

sans
serif
script
mono / monospace

Scale & Line Height

Clone provides a built in means of controlling a typography scale based on a ratio. The system sets the base font to the user's default browser setting to begin with. By default, Clone uses a 1.333 scale for determining the size of the typography. This can be changed by modifying the $clone-font-scale variable. This scale appears as follows:

Captions & Labels

Please note that this scale is not automatically applied to the heading elements and must be applied manually using the data-c-font-size="SIZE" property. This is so that you can manually determine the appearance of a heading when the markup structure might dictate otherwise.

# SIZE

h1
h2
h3
h4
h5
normal / regular
small

Line height defaults to 1.5, but can manually changed using the $clone-line-height variable.

Font Colour

<div data-c-colour="COLOUR"></div>, <div data-c-color="COLOR"></div>

This attribute allows you to assign a font colour to an element and all of its children.

Available Arguments

# COLOUR

white
gray / grey
black
c1 (Theme Colour 01)
c2 (Theme Colour 02)
c3 (Theme Colour 03)
c4 (Theme Colour 04)
c5 (Theme Colour 05)
stop
slow
go

Font Weight

<div data-c-font-weight="WEIGHT"></div>

This attribute allows you to assign a font weight to an element.

Available Arguments

# WEIGHT

100
200
300
400
normal
500
600
700
bold
800
900

Font Style

<div data-c-font-style="STYLE"></div>

This attribute allows you to assign a text style to an element. You can use <div data-c-font-weight="bold"></div> (found below) to bold items instead.

Available Arguments

# STYLE

italic
underline

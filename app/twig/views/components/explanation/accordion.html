The group argument is used to define a group of accordions and is required even in the event that a single accordion exists. The alt argument can be used to force the accordion group to alternate between white and the assigned colour set in the variable {% filter escape %}$clone-accordion-alternating-background-colour{% endfilter %}. If alt is omitted you can use {% filter escape %}data-c-background{% endfilter %} to assign a custom background colour to the {% filter escape %}data-c-accordion="trigger"{% endfilter %} element.
